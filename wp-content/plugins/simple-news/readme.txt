== Description ==

A very simple news plugin that output an unordered list. (img, title and link)

* News are order by pub date.
* A post type "NEWS" is added.
* A widget show the 5 latest news.
* Shortcodes: [news] - [news number="2"] - [news order="desc"] - [news number="3" order="asc"] - [news cat="1"] - [news cat="1,2"]
* A Danish translation is included.

== Installation ==

1. Download the plugin, and unzip it.
2. Place the News folder in your wp-content/plugins folder.
3. Activate the plugin from the plugins tab of your Wordpress admin.
4. Save permalink structure again - because of the post type (news)
